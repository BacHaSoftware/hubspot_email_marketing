Hubspot Integration

A product of Bac Ha Software allows to Synchronized data between Hubspot and Odoo, provides comprehensive solutions to email marketing and related problems.

Set up

Before installing this module, you need to install libraries simplejson, hubspot, hubspot-api-client, boto3 :
pip3 install simplejson
pip3 install hubspot
pip3 install hubspot-api-client
pip3 install boto3

And this module also depends on our other module: Mass Mailing (bhs_mass_mailing)

Featured Highlight

Data Synchronization

Hubspot data and odoo mailing contacts are synchronized daily.

Simple Setup

After install module, just need to input HubSpot API Key at General Settings.

Key Features

Hubspot Contact And Companies

Synchronize contact and company data between Hubspot and Odoo daily. Synchronized data includes: name, city, state, country, industry...

Add To Blacklist

Automatically add to email blacklist Suppressed emails on SES and delete their mailing contacts.

Update Unqualified Contacts On Hubspot

Automatically update contacts on hubspot to UNQUALIFIED when they are blacklist on Odoo (run schedule action once a day, after retrieve BOUNCE and COMPLAIN emails from SES and add to mail blacklist).

Mailing Contact Features

Block duplicate mailing contacts, import mailing contact data into mailing list, add to blacklist feature.

Blacklist Domain

Add model to record blacklist domains. Block import emails belonging to blacklist domains (edu,gov,org,....).

Add And Remove From Old List

Add mailing contacts to a list and delete them from their old lists.

Mailing Contact Contacted

Record mailing contacts that no longer want to change their mailing list (example: current customers). The last purpose is to no longer send marketing emails to these mailing contacts.
